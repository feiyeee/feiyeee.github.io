棋盘问题

51. N皇后

按照国际象棋的规则，皇后可以攻击与之处在同一行或同一列或同一斜线上的棋子。

n 皇后问题 研究的是如何将 n 个皇后放置在 n×n 的棋盘上，并且使皇后彼此之间不能相互攻击。

给你一个整数 n ，返回所有不同的 n 皇后问题 的解决方案。

每一种解法包含一个不同的 n 皇后问题 的棋子放置方案，该方案中 'Q' 和 '.' 分别代表了皇后和空位。

示例 1：


输入： n = 4
输出： [[".Q..","...Q","Q...","..Q."],["..Q.","Q...","...Q",".Q.."]]
**解释：**如上图所示，4 皇后问题存在两个不同的解法。

示例 2：
输入： n = 1
输出： [["Q"]]

提示：

  • 1 <= n <= 9

public List Array2List(char[][] chessboard) {
    List<String> list = new ArrayList<>();
    
    for (char[] c : chessboard) {
        // 把char转换成String之后加入到list里
        list.add(String.copyValueOf(c));
    }
    return list;
}

再附上树形结构图：

for (int col=0; col<n; col++) {

合法条件：

  1. 不能同行
  2. 不能同列
  3. 不能同斜线
public boolean isValid(int row, int col, char[][] chessboard, int n) {
    // 不用判断同行，具体可以看树形结构图
    // 是否同列
    for (int i=0; i<row; i++) {
        if (chessboard[i][col] == 'Q') {
            return false;
        }
    }
    
    // 判断45°方向
    for (int i=row-1, j=col-1; i>=0 && j>=0; i--, j--) {
        if (chessboard[i][j] == 'Q') {
            return false;
        }
    }
    
    // 判断135°方向
    for (int i=row-1, j=col+1; i>=0 && j<n; i--, j++) {
        if (chessboard[i][j] == 'Q') {
            return false;
        }
    }
    
    return true;
}

整合答案

class Solution {
    List<List<String>> res = new ArrayList<>();

    public List<List<String>> solveNQueens(int n) {
        char[][] chessboard = new char[n][n];
        for (int i=0; i<n; i++) {   // 初始化
            Arrays.fill(chessboard[i], '.');
        }
        backTracking(n, 0, chessboard);
        return res;
    }

    public void backTracking(int n, int row, char[][] chessboard) {
        // 终止条件
        if (row == n) {
            res.add(Array2List(chessboard));
            return;
        }

        for (int col=0; col<n; col++) {
            if (isValid(row, col, chessboard, n)) {
                chessboard[row][col] = 'Q';
                backTracking(n, row+1, chessboard);
                chessboard[row][col] = '.';
            }
        }
    }

    public List<String> Array2List(char[][] chessboard) {
        List<String> list = new ArrayList<>();
        for (char[] c : chessboard) {
            list.add(String.copyValueOf(c));
        }
        return list;
    }

    public boolean isValid(int row, int col, char[][] chessboard, int n) {
        // 列
        for (int i=0; i<row; i++) {
            if (chessboard[i][col] == 'Q') {
                return false;
            }
        }

        // 45°
        for (int i=row-1, j=col-1; i>=0 && j>=0; i--, j--) {
            if (chessboard[i][j] == 'Q') {
                return false;
            }
        }

        // 135°
        for (int i=row-1, j=col+1; i>=0 && j<n; i--, j++) {
            if (chessboard[i][j] == 'Q') {
                return false;
            }
        }

        return true;
    }
}

37. 解数独

编写一个程序，通过填充空格来解决数独问题。

数独的解法需 遵循如下规则：

  1. 数字 1-9 在每一行只能出现一次。
  2. 数字 1-9 在每一列只能出现一次。
  3. 数字 1-9 在每一个以粗实线分隔的 3x3 宫内只能出现一次。（请参考示例图）

数独部分空格内已填入了数字，空白格用 '.' 表示。

问题分析

树形结构上：

和 51. N皇后 不同，上题只需要遍历每一行，每一行寻找一个合法的位置放置皇后即可；而这题是需要遍历每一行中的每一个位置，判断当前位置填入哪个数字是合法的

上回溯三部曲：

1、确定回溯函数模版以及返回参数

首先，回溯函数的返回值应该是bool类型

  • 解数独找到一个符合的条件，就立刻返回
  • public boolean backTracking(char[][] board)

2、确定回溯函数终止条件

本题不需要终止条件，因为解数独要遍历整个树形结构寻找可能的叶子节点就立刻返回

3、分析回溯搜索的遍历过程

public boolean backTracking(char[][] board) {
    for (int i=0; i<board.length; i++) {    // 遍历行
        for (int j=0; j<board.length; j++) {    // 遍历列
            if (board[i][j] != '.') continue;   // 有数
            for (char k='1'; k<='9'; k++) {
                if (isValid(i, j, k, board)) {    // 如果合法，直接放置k
                    board[i][j] = k;
                    if (backTracking(board)) return true;    // 找到一组合适的，立刻返回
                    board[i][j] = '.';    // 回溯
                }
            }
            return false;    // 9个数都用完了，返回false
        }
    }
    return true;
}
public boolean isValid(int row, int col, char val, char[][] board) {
    // 合法条件
    //    1、同行不重复
    //    2、同列不重复
    //    3、9宫格内不重复
    
    // 判断行
    for (int i=0; i<9; i++) {
        if (board[row][i] == val) return false;
    }
    
    // 判断列
    for (int j=0; j<9; i++) {
        if (board[j][col] == val) return false;
    }
    
    // 判断9宫格
    int startRow = (row / 3) * 3;
    int startCol = (col / 3) * 3;
    for (int i=startRow; i<startRow+3; i++) {
        for (int j=startCol; j<startCol+3; j++) {
            if (board[i][j] == val) return false;
        }
    }
    return true;
}

整合答案

class Solution {
    public void solveSudoku(char[][] board) {
        backTracking(board);
    }

    public boolean backTracking(char[][] board) {
        // 遍历行
        for (int i=0; i<board.length; i++) {
            // 遍历列
            for (int j=0; j<board[0].length; j++) {
                if (board[i][j] != '.') continue;
                for (char k='1'; k<='9'; k++) {
                    if (isValid(i, j, k, board)) {
                        board[i][j] = k;
                        if (backTracking(board)) return true;
                        board[i][j] = '.';
                    }
                }
                return false;
            }
        }
        return true;
    }

    public boolean isValid(int row, int col, char val, char[][] board) {
        // 判断行
        for (int i=0; i<9; i++) {
            if (board[i][col] == val) return false;
        }

        // 判断列
        for (int j=0; j<9; j++) {
            if (board[row][j] == val) return false;
        }

        // 判断9宫格
        int startRow = (row / 3) * 3;
        int startCol = (col / 3) * 3;
        for (int i=startRow; i<startRow+3; i++) {
            for (int j=startCol; j<startCol+3; j++) {
                if (board[i][j] == val) return false;
            }
        }
        return true;
    }
}

尝试总结

棋盘问题是二维回溯问题

对于二维问题，可以对题目中的每一个格子都进行分析：

  1. 可以放入什么？
  2. 什么时机放入是合法的？

而对每一个格子进行分析，也就是用二层循环，可以理解为纯暴力（毕竟回溯就是纯暴力）

还有，对于回溯函数的返回值，之前遇到的都是 void，但对于只需要得到一个答案的问题，就可以控制返回值为 boolean 或者其它（根据题意），比如 37. 解数独

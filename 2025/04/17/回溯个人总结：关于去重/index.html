回溯个人总结：关于去重

关于前面做到的需要去重的回溯问题：

40. 组合总和 II

90. 子集 II

491. 非递减子序列

47. 全排列 II

对比题目可以发现，需要去重的问题有以下几个特点：

  1. 给出的输入集合包含重复元素
  2. 输入集合的顺序不重要（可以重新排序）
  3. 输出的集合不能包含重复的集合

去重的两种方式：

  1. used[] 数组
  2. 自己控制 index 进行去重

以 40. 组合总和 II 举例：

1、used[] 数组

我们需要去重的是：树里面同一层出现的重复元素

所以当 candidates[i-1]==candidates[i] && used[i-1]==0 时，直接continue。当然，还需要加上 i>0 这个条件

  • 为什么是 used[i-1] == 0？
    • 因为我们要比较的是同一层的上一个节点 -> 也就是回溯回去后得到的节点，这时候我们已经把 used[i-1] 置为0了，所以应该是 used[i-1] == 0
    • 如果我们需要找的是上一层的上一个节点，那么应该是 used[i-1] == 1
class Solution {
    List<List<Integer>> res = new ArrayList<>();
    List<Integer> list = new ArrayList<>();
    int[] used;
    
    public List<List<Integer>> combinationSum2(int[] candidates, int target) {
        used = new int[candidates.length];
        Arrays.fill(used, 0);
        // [1,1,2,5,6,7,10]
        Arrays.sort(candidates);
        backTracking(candidates, target, 0, 0);
        return res;
    }

    public void backTracking(int[] candidates, int target, int index, int sum) {
        if (target == sum) {
            res.add(new ArrayList<>(list));
            return;
        }

        for (int i=index; i<candidates.length; i++) {
            // 去重
            if (i>0 && candidates[i-1]==candidates[i] && used[i-1]==0) {
                continue;
            }
            // 剪枝
            if (sum + candidates[i] > target) break;
            list.add(candidates[i]);
            used[i] = 1;
            backTracking(candidates, target, i+1, sum+candidates[i]);
            list.remove(list.size()-1);
            used[i] = 0;
        }
    }
}

2、用index控制去重时机

i > index 时也就是说当前遍历的节点不是根节点，那么就可以直接判断该节点是否和上一个节点一样，如果一样，直接continue

这个比 used[] 数组方便很多：

class Solution {
    List<List<Integer>> res = new ArrayList<>();
    List<Integer> list = new ArrayList<>();
    
    public List<List<Integer>> combinationSum2(int[] candidates, int target) {
        // [1,1,2,5,6,7,10]
        Arrays.sort(candidates);
        backTracking(candidates, target, 0, 0);
        return res;
    }

    public void backTracking(int[] candidates, int target, int index, int sum) {
        if (target == sum) {
            res.add(new ArrayList<>(list));
            return;
        }

        for (int i=index; i<candidates.length; i++) {
            // 去重
            if (i>index && candidates[i-1]==candidates[i]) {
                continue;
            }
            // 剪枝
            if (sum + candidates[i] > target) break;
            list.add(candidates[i]);
            backTracking(candidates, target, i+1, sum+candidates[i]);
            list.remove(list.size()-1);
        }
    }
}

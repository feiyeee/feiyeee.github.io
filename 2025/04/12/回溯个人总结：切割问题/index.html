131. 分割回文串

给你一个字符串 s，请你将 s 分割成一些 子串，使每个子串都是 回文串 。返回 s 所有可能的分割方案。

示例 1：
输入： s = "aab"
输出： [["a","a","b"],["aa","b"]]

示例 2：
输入： s = "a"
**输出： **[["a"]]

提示：

  • 1 <= s.length <= 16
  • s 仅由小写英文字母组成

问题分析

我们可以把字符串 s 中的每一个字母后面都添加一个逗号，那么问题就转换成了 ”选或不选“ 的问题：（图中的 ”。“ 表示待选择的位置）

关于回溯函数的参数、终止条件都很常规，这里不赘述

我们主要来探究回溯循环该怎么写

可以参照上图，如果我们选择了这个逗号，就判断这样分割之后这个子串是不是回文串

  • 如果是，就选，加入到list，继续回溯
  • 如果不是，就不选，直接跳过

附上：

for (int i=index; i<s.length(); i++) {
    // 如果截取后是回文，选，加入list里
    if (isHuiwen(s, index, i)) {
        list.add(s.substring(index, i+1));
    } else {
        continue;
    }
    backTracking(s, i + 1);
    list.remove(list.size()-1);
}

完整答案

class Solution {
    List<List<String>> res = new ArrayList<>();
    List<String> list = new ArrayList<>();

    public List<List<String>> partition(String s) {
        backTracking(s, 0);
        return res;
    }

    public void backTracking(String s, int index) {
        // 终止条件
        if (index == s.length()) {
            res.add(new ArrayList<>(list));
            return;
        }

        // 循环
        for (int i=index; i<s.length(); i++) {
            // 如果截取后是回文，选，加入list里
            if (isHuiwen(s, index, i)) {
                list.add(s.substring(index, i+1));
            } else {
                continue;
            }
            backTracking(s, i + 1);
            list.remove(list.size()-1);
        }
    }

    public boolean isHuiwen(String s, int start, int end) {
        while (start < end) {
            if (s.charAt(start) != s.charAt(end)) {
                return false;
            }
            start++;
            end--;
        }
        return true;
    }
}

93. 复原IP地址

有效 IP 地址 正好由四个整数（每个整数位于 0 到 255 之间组成，且不能含有前导 0），整数之间用 '.' 分隔。

  • 例如："0.1.2.201" 和 "192.168.1.1" 是 有效 IP 地址，但是 "0.011.255.245"、"192.168.1.312" 和 "192.168@1.1" 是 无效 IP 地址。

给定一个只包含数字的字符串 s ，用以表示一个 IP 地址，返回所有可能的有效 IP 地址，这些地址可以通过在 s 中插入 '.' 来形成。你 不能 重新排序或删除 s 中的任何数字。你可以按 任何 顺序返回答案。

示例 1：
输入： s = "25525511135"
输出： ["255.255.11.135","255.255.111.35"]

示例 2：
输入： s = "0000"
输出： ["0.0.0.0"]

示例 3：
输入： s = "101023"
输出： ["1.0.10.23","1.0.102.3","10.1.0.23","10.10.2.3","101.0.2.3"]

提示：

  • 1 <= s.length <= 20
  • s 仅由数字组成

这个同上题，把每个数字后面都加一个伪 "."，将其转换成 “选或不选” 问题

不同的是，这个题还有一个限制条件，就是只能分割成四组数字，也就是说，有且仅有三个 "."

而且这里对字符串 s 原地做改变，比用 StringBuffer 要方便很多，所以对于最后一组的选择，我们实际上是不需要加上 "." 的，所以把这个的合法条件放在循环里判断也很不方便，直接在终止条件里判断即可

答案附上：

class Solution {
    List<String> res = new ArrayList<>();

    public List<String> restoreIpAddresses(String s) {
        backTracking(s, 0, 0);
        return res;
    }

    public void backTracking(String s, int index, int pointNum) {
        if (pointNum == 3) {
            if (isValid(s, index, s.length()-1)) {
                res.add(s);
            }
            return;
        }

        for (int i=index; i<s.length(); i++) {
            if (isValid(s, index, i)) {
                s = s.substring(0, i+1) + "." + s.substring(i+1);
                pointNum++;
                backTracking(s, i+2, pointNum);
                s = s.substring(0, i+1) + s.substring(i+2);
                pointNum--;
            } else {
                break;
            }
        }
    }

    // 控制的条件：
    //   1、只能有三个 "."
    //   2、每个整数在0~255之间
    public boolean isValid(String s, int i, int j) {
        if (i > j) {
            return false;
        }
        if (s.charAt(i)=='0' && i!=j) {
            return false;
        }
        int num = 0;
        for (int m=i; m<=j; m++) {
            if (s.charAt(m) < '0' || s.charAt(m) > '9') {
                // 非法数字
                return false;
            }
            num = num*10 + (s.charAt(m)-'0');
            if (num > 255) {
                return false;
            }
        }
        return true;
    }
}
